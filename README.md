# 🚀 AI Resume Analyzer

## 📌 Project Description

This is a **Python Full Stack Web Application** that analyzes resumes using **Machine Learning (Cosine Similarity)** and compares them with job descriptions.

---

## ✨ Features

* 🔐 User Login & Registration
* 📄 Resume Upload (PDF)
* 🤖 AI-based Resume Scoring
* 📊 Missing Skills Detection
* 🗂️ History Tracking (User-wise results)

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS
* SQLite Database
* NLP (spaCy)
* Machine Learning (scikit-learn)

---

## 📸 How It Works

1. User logs in / registers
2. Uploads resume
3. Enters job description
4. System calculates similarity score
5. Displays missing skills
6. Saves result in history
